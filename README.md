# Soyuz Docs

Documentação oficial da linguagem de programação Soyuz, construída com Astro e Starlight.

## Requisitos

- Node.js
- pnpm

## Desenvolvimento

Instale as dependências:

```sh
pnpm install
```

Inicie o servidor local:

```sh
pnpm dev
```

Gere o build de produção:

```sh
pnpm build
```

O conteúdo principal fica em `src/content/docs/`.
